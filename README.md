## 🎮 About the Project
**Guess the Brainrot** is an interactive web-based puzzle game disguised as an administrator login screen. Inspired by concepts like *The Password Game*, users must type a password that satisfies an ever-growing list of pop-culture, gaming, and internet-slang "Security Policies." 

Fail a rule? You'll be locked out and forced to beat a random, high-stress mini-game to get back in. This project embraces absolute chaos to test your internet literacy and gaming reflexes.

## ✨ Features
- **Built for Speed (and Chaos):** 20 unique, fully playable penalty mini-games coded in vanilla JS within a 6-hour window.
- **Dynamic Security Policies:** 25 randomized trivia and meme rules ranging from Gen-Z slang to classic gaming lore.
- **Glassmorphism UI:** Beautiful, modern Tailwind CSS styling with backdrop blurs, 3D buttons, and animated gradients.
- **Custom Audio Engine:** Includes background music and a variety of situational sound effects (SFX) that react to your gameplay.
- **Zero Dependencies:** Runs entirely in the browser without Node.js, Webpack, or any backend servers.

## 🕹️ The Penalty Mini-Games
If you trigger a security alert, you'll be thrown into one of 20 randomized challenges, including:
- **Geometry Dash:** Jump over the approaching spike.
- **Osu! / Aim Lab:** Click the targets before they disappear.
- **Wild Encounter:** Battle a wild Charizard in a retro turn-based RPG UI.
- **Defusal:** Cut the colored wires in the exact right order.
- **Terminal Override:** Type the hacker code before the countdown ends.
- **Steady Hand:** Guide a block through a maze without touching the walls.
- **Whack-A-Mole:** Squish 5 moles before they hide.
- **Vibe Check:** Hit the spacebar exactly when the rhythm cursor hits the green zone.
- *...and 12 more chaotic games!*

## 🚀 Getting Started

Because this project is built with standard web technologies and utilizes a CDN for styling, installation is completely frictionless.

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge).
- An active internet connection (required to load Tailwind CSS, images, and audio files).

### Installation
1. Clone the repository:
   ```bash
   git clone [https://github.com/yourusername/guess-the-brainrot.git](https://github.com/yourusername/guess-the-brainrot.git)
   ```
2. Navigate to the project directory:
   ```bash
   cd guess-the-brainrot
   ```
3. Open the `index.html` file directly in your browser. 

## 🛠️ Tech Stack
- **HTML5** (Structure & Canvas-less rendering)
- **CSS3 / Tailwind CSS via CDN** (Styling & Animations)
- **Vanilla JavaScript** (Logic, DOM manipulation, and Game Loops)
